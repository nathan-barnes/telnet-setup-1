## Installing deps

```bash
yarn install
```

## Setting env vars

Required:

```bash
export ZEA_TELNET_HOST=127.0.0.1
export ZEA_TELNET_PORT=23
export ZEA_COLLAB_ROOM=surveying-machine
```

Optional:

```bash
export ZEA_STREAMER_TYPE=mock
export ZEA_STREAMER_TYPE=telnet
export DEBUG=zea:*
```

## Running

```bash
yarn run start
```
